豆瓣电影vue spa
====================
### 项目说明
公司项目刚做完有点空余时间，平时喜欢看电影所以写了一个小小的豆瓣电影
，
搜索提示因为豆瓣接口请求太多次ip就会被封，所以输入完之后要等一秒左右之后才会有提示，所有接口均使用豆瓣接口，仅供学习交流使用

### 所用技术栈:
* vue全家桶
* mint ui
* jsonp
* animate.css

### 启动项目
建议用淘宝镜像cnpm安装依赖，npm太慢

安装依赖 `cnpm install`

启动服务器和热加载 `cnpm run dev`

打包 `cnpm run build`

当有空余时间时和豆瓣开放更多api时会持续更新
